## Overview
This is a simple Windows desktop application built with WPF.  
It automates Windows activation using generic keys and KMS tickets.  

- Installs a generic key for your Windows edition.
- Copies and applies a KMS ticket.
- Runs `clipup.exe` to finalize activation.
- Shows success or error messages to the user.

## Usage
1. Run the app **as Administrator**.
2. Make sure the `Tickets\KMS38.xml` file exists in the application directory.
3. Click the activation button in the app.
4. The app will show a message box with success or failure.
